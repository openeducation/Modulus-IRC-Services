modes(modes) click to toggle source

Called when a MODE message is received for this user. Updates the array of modes by parsing the mode update string to find +/- changes.

Parameter is the mode change string (not an origin info object). 

    # File includes/user.rb, line 60
60:     def modes(modes)
61:       plus = true
62: 
63:       modes.each_char { |c|
64:         if c == "+"
65:           plus = true
66:         elsif c == "-"
67:           plus = false
