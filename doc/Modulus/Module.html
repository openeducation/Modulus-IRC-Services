new(name, description, author, version) click to toggle source

Initialize a new module object. Not to be confugused with Ruby modules, this refers to Modulus modules, such as NickServ.

Parameters include the name of the module, a brief description of the module, the author of the module, adn the version of the module.

A hash table of module commands is initialized here.
