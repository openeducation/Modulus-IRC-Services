new(config) click to toggle source

Create a new logger object. The configuration object is the only parameter and is used during initializiation.

If the configured log directory does not exist, attempt to create it, or die trying (fatal error).

If a log channel is configured, we’ll use that for on-protocol logging. 

    # File includes/logging.rb, line 43

44:       logdir = config.getOption('Core', 'log_location')
45:       @logChannel = config.getOption('Core', 'log_channel')
46: 
47:       @chatVerbosity = self.verbToNum(config.getOption('Core', 'log_channel_verbosity')) # Logging to log chan
48: 
49:       unless Dir.exists? logdir
50:         begin    
51:           Dir.mkdir logdir
52:         rescue => e
53:           $stderr.puts "Fatal error: Cannot create log directory #{logdir}:"
54:           $stderr.puts "#{e}"

57:       end

logToChannel(severity, msg) click to toggle source

Send the log message to IRC on the log channel, if configured.

The parameters are the serverity (one of our constants) and the log message.

If no log channel is configured or the link is not active, abort. 

     # File includes/logging.rb, line 112
