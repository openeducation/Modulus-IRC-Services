event(event, *args) click to toggle source

Fire the callback for the given event. Optionally, one or more additional parameters may be included to be passed to the callback function. If the function doesn’t accept the given number of parameters, an exception may be thrown. 

    # File includes/events.rb, line 48

53:         $log.debug 'events', "Attempted to fire event with no recipients: #{event}"
54:       end

register(event, sender, func) click to toggle source

Register a new event with callback. The parameters are the event for which we are registering (such as :join), the sender of the function call (used as the callback function name owner) and the name of the function to use as callback.
